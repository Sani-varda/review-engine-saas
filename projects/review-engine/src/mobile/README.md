# Review Engine Mobile App

The mobile companion app for The Review Engine.

## Features
- **Secure Authentication**: Traditional login and Google Sign-In integration.
- **Dynamic Dashboard**: Real-time stats and reputation overview.
- **Review Management**: View, track, and reply to Google Reviews.
- **AI-Powered Replies**: Generate professional responses using the backend AI engine.
- **Review Requests**: Quickly send request invites to customers via email/SMS.

## Setup
1. Ensure Flutter is installed.
2. Run `flutter pub get`.
3. Update `lib/services/api_service.dart` with your backend URL.
4. Run `flutter run`.

## Tech Stack
- **Flutter**: Cross-platform mobile development.
- **Provider**: State management.
- **Lucide Icons**: Modern icon set.
- **Google Fonts**: Premium typography (Inter).
